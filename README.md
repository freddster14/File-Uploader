# File-Uploader
Live Preview: [Click Here](https://file-uploader-p3q0.onrender.com)

## What it is  
Nuvio is a full-stack file storage and sharing web application built with Node.js and Express. It allows users to securely upload, organize, preview, and share files and folders through permission-controlled links, with a clean and responsive interface.
## Features  
- User registration & authentication 
- Authorization checks on all protected routes
- Create, edit, and delete posts (for authenticated users)  
- MVC structure (controllers, models, views)
- Share entire folders (including nested content)
- Generate secure, token-based share links  
- Responsive UI

## Tech Stack  
- **Backend:**  Node.js, Express, Prisma ORM, PostgreSQL, Passport.js, Express-session, bcrypt 
- **Frontend:** HTML, CSS, EJS 
- **Authentication:** Express-session, bcrypt, Passport.js
